Get Started

You can use our custom CSS Components by linking the following code in your HTML. 

  
  <link
    rel="stylesheet"
    href="https://comp-libr.netlify.app/styles/components.css"
  />

Also, link this JavaScript file at the end of the body for certain components to work properly. 

  
  <script src="https://comp-libr.netlify.app/scripts/script.js"></script>
